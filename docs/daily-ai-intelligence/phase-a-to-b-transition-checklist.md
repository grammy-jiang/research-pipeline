# Phase A to Phase B Transition Checklist

Before starting Phase B, verify:

- [ ] A01-A12 are `audit_pass`.
- [ ] Phase A proof packs exist.
- [ ] Phase A acceptance contracts exist.
- [ ] Phase A normal-day offline e2e test passes.
- [ ] Phase A low-signal-day offline e2e test passes.
- [ ] Phase A no-news-day offline e2e test passes.
- [ ] Phase A report validator passes.
- [ ] `research-pipeline brief run` produces the fixed artifact layout.
- [ ] `phases.A.status` is `complete`.
- [ ] `phases.B.status` is `in_progress`.

If any item fails, do not start Phase B.

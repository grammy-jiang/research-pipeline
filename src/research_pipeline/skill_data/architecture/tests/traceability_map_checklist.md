# Checklist: Traceability Map

- [ ] A blueprint-to-architecture traceability map exists (table form).
- [ ] Each major architecture section names at least one blueprint source (or
      an explicit "not present in blueprint" note).
- [ ] Every major architecture decision traces to one of: a blueprint section,
      a user clarification, a rule-pack decision, or an explicit recorded
      assumption.
- [ ] Absent blueprint sections are marked, not invented.
- [ ] The map is consistent with the sections actually produced (no dangling
      references to sections that do not exist).
